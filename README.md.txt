 O projeto foi desenvolvido para realizar o controle de estoque de uma adega de vinhos.

O objetivo é realizar um controle eficaz e conciso dos vinhos, trazendo informações como:

- Lista geral de vinhos;
- Lista para inserir vinhos especias;
- Inclusões e remoções;
- Informações relevantes sobre o estoque;

Linguagem C*;

Bibliotecas usadas:

#include <stdio.h>
#include <stdlib.h>

O programa deve ser compilado no Replit;

https://replit.com
